# Runoflow Landing Page

A modern, minimal landing page for Runoflow — AI Sales Systems & Deal Flow Infrastructure.

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **TypeScript**

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

1. Push this repo to GitHub/GitLab/Bitbucket.
2. Import the project in [Vercel](https://vercel.com).
3. Vercel will auto-detect Next.js and set the build command.
4. Deploy.

Or use the [Vercel CLI](https://vercel.com/cli):

```bash
npm i -g vercel
vercel
```

## Project Structure

- `app/` — App Router pages and layout
- `components/` — Reusable sections (Navbar, Hero, Problem, Solution, etc.)
- `app/globals.css` — Global styles and Tailwind
